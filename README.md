## About the Project

Nutritionist is a web application designed to empower individuals in their journey towards better health through balanced, enjoyable, and sustainable nutrition. With a team of qualified nutritionists and evidence-based strategies, our mission is to support you in making lasting, meaningful changes to your diet and overall well-being.

## Features

Personalized Nutrition Plans: Tailored to meet individual health goals and dietary preferences.
Expert Guidance: Insights and support from qualified nutritionists and dietitians.
Sustainable Approach: Focus on balanced nutrition, whole foods, and mindful eating.
Community Support: Join a community dedicated to transforming their lives through better nutrition.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js: Ensure you have Node.js installed, which is required to run this React project.
- npm: npm is used to manage dependencies; it typically comes with Node.js.

## Installation

1. Clone the repository:

   ```
   git clone https://github.com/your-username/Nutritionist.git
   ```

2. Navigate to the project directory:

   ```
    cd Nutritionist
   ```

3. Install the dependencies:

   ```
   npm install
   ```

# Usage

To start the application in development mode:

    npm start

Open http://localhost:3000 to view it in the browser. The page will reload if you make edits, and you will see any lint errors in the console.

# Technologies

This project uses the following technologies and libraries:

- React: ^18.3.1
- Redux Toolkit: For state management
- React Router DOM: For routing
- Font Awesome: For icons
- Tailwind CSS: For styling
- Swiper: For carousels and slides
- Testing Library: For testing components
- GitHub Pages: For deployment

# Deployment

The project is automatically deployed to GitHub Pages whenever changes are pushed to the main branch. You can view the live project at: Nutritionist Live

To deploy manually, you can use the following command:

    npm run deploy
